# ya_sam_api_tests_review
# Тесты API на проверку сценария "Получить информацию о заказе по его номеру track" после успешного создания заказа Яндекс Самокат
- Для запуска тестов должны быть установлены пакеты pytest и requests
- Запуск всех тестов выполняется командой pytest или python -m pytest
- Перед запуском в файле config.py необходимо актуализировать параметр id в url https://{id}.serverhub.praktikum-services.ru/
- Запросы для проверки работы БД (1.вывод списка логинов курьеров с количеством их заказов в статусе «В доставке» (поле inDelivery = true) и 2.вывод всех трекеров заказов и их статусов) находятся в файле data_base_requests.sql
