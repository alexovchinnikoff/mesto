### **Тестирование приложения для путешественников Mesto.**

В учебных задач практиковался с техниками тест-дизайна и в составлении тестовой документации для [фронтенда](https://docs.google.com/spreadsheets/d/1fVGpFCmSKFs6Xr7uLqv4IXET4o24uDMi/edit?usp=drive_link&ouid=104642614711975961739&rtpof=true&sd=true) и [API](https://docs.google.com/spreadsheets/d/18z4XkCQRvgHABOfnsdEQn31ythePD1CD/edit?usp=drive_link&ouid=104642614711975961739&rtpof=true&sd=true).
Выбрал это приложения для первого опыта в автоматизации тестовых сценариев для фронтенда.
Вот примеры автоматизации двух сценариев:
1. [Смена аватара и возврат прежнего в профиле пользователя]().
2. [Добавление новой карточки]().
тесты находятся tests/test_cases

Для запустка необходимо установить Selenium Webdriver.
